# Site de compras
## Hamburgueria

O site visa a compra de lanches disponiveis na vitrine com possibilidade de adição ao carrinho ao se cadastrar ou fazer o login.

* React.Js, Typescript, Context API, Chakra UI, React-icons, React-router-dom
